                                        │
                         ┌──────────────┴──────────────┐
                         │                             │
                    Frontend                      FastAPI
                                                       │
                              ┌────────────┬───────────┴──────────┐
                              │            │                      │
                         PostgreSQL      Redis              Cloudinary
                              │
                        Backup diário

<h2 align="center">📚 Roadmap atual</h2>

| Módulo | Tópicos principais | Projeto prático |
|:--|:--|:--|
| Linux | arquivos, usuários, permissões, serviços, UFW, Fail2Ban | Configurar VPS Ubuntu |
| Git & GitHub | branches, merge, pull request | Publicar projetos |
| Docker | Dockerfile, images, containers, volumes | Containerizar API FastAPI |
| Docker Compose | orquestração multi-serviço | FastAPI + PostgreSQL + Redis + Nginx |
| PostgreSQL | SQL, modelagem, backup, restore | Sistema de produtos |
| FastAPI | REST, JWT, Alembic | Loja virtual |
| Nginx | reverse proxy, HTTPS, cache | Publicar domínio |
| Redis | cache, sessões, filas | Cache de produtos |
| Segurança | hash de senha, JWT, variáveis de ambiente | Hardening da VPS |
| CI/CD | GitHub Actions, build e deploy | Pipeline automatizado |
| Monitoramento | Uptime Kuma, Grafana, Prometheus | Observabilidade |
| Backup & DR | pg_dump, restore, recuperação de VPS | Simular disaster recovery |
